# Feasibility Study

## Market Feasibility

| Parameter | Estimate |
|---|---|
| Total Addressable Market | 2.3 million households |
| Serviceable Market | 150,000 households |
| Initial Target (Yr 1-3) | 5,000 members |

## Technical Feasibility

- **Core Banking System**: Mambu (cloud-native)
- **Mobile App**: React Native (iOS & Android)
- **USSD**: Africa's Talking API
- **Payments**: M-Pesa Daraja API integration

## Financial Feasibility

| Year | Revenue (KES) | Costs (KES) | Net (KES) |
|---|---|---|---|
| Year 1 | 3,500,000 | 5,200,000 | -1,700,000 |
| Year 2 | 7,800,000 | 6,500,000 | 1,300,000 |
| Year 3 | 14,200,000 | 8,800,000 | 5,400,000 |

Break-even is projected at **Month 18** of operations.
